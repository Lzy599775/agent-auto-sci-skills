kdense-ml-ai-selected

从 K-Dense Scientific Agent Skills 中选择性封装的机器学习与人工智能工具包。它补强技术路线说明，但在 Auto-sci-research 中仍遵守地理学、体育学、城市暴露和健康研究的证据边界。

Selected K-Dense ML/AI wrapper for technical modeling workflows, used under Auto-sci-research evidence and domain boundaries.

包含什么

传统与深度学习

scikit-learn、pytorch-lightning、transformers。

解释与结构模型

shap、torch-geometric、umap-learn。

时间序列

aeon 与 timesfm-forecasting，用于时序分类、预测和趋势建模。

本地适配

面向空间验证、城市迁移、健康影响解释、文本筛选/编码和可复现论文写作。

什么时候调用

当任务涉及模型训练、文本/图像/表格建模、SHAP/XAI、GNN、时序预测、降维聚类、城市迁移或模型结果写进论文时调用。

使用 kdense-ml-ai-selected，帮我设计体育设施可达性与健康行为模型。
要求先给 baseline，再考虑复杂模型，并检查空间泄露、城市迁移、SHAP 解释边界和论文写法。

Codex 部署

git clone https://github.com/Lzy599775/agent-auto-sci-skills.git
cd agent-auto-sci-skills
.\scripts\install.ps1

安装后重启 Codex。推荐与 agent-auto-sci-ai-ml、agent-auto-sci-methodology、agent-auto-sci-geospatial 联合使用。

边界

  • 不把随机切分作为空间数据默认验证方式。
  • 不把 SHAP、特征重要性或 embedding 直接写成因果机制。
  • 不在没有 baseline 的情况下直接堆复杂模型。
  • 不把私有健康、位置或未发表数据发送到外部 API。
