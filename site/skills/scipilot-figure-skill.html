scipilot-figure-skill

从 Haojae/scipilot-figure-skill 本地封装的科学图件顾问。它关注图件论点、图型选择、排版安全、中文/英文标签、视觉层级和视觉自检闭环。

Locally packaged SciPilot Figure Skill for figure arguments, chart choice, layout safety, CJK/English labels, visual hierarchy, and visual QA loops.

核心能力

从论点选图型

先明确图件要证明什么，再选择散点、地图、热图、网络图、机制图或多面板组合。

视觉自检闭环

检查标签、图例、单位、坐标轴、留白、对比度、字体和导出格式。

中文标签友好

适合中文研究过程和英文投稿图件并存的场景，提醒 CJK 字体与排版风险。

论文叙事对齐

把 figure、caption、结果段落和 claim-evidence map 对齐。

什么时候调用

使用 scipilot-figure-skill，帮我重设计体育设施可达性与绿色暴露论文的 Figure 1-5。
请先列出每张图要支撑的 claim，再给图型、视觉编码、面板布局、caption 和视觉 QA 清单。

Codex 部署

git clone https://github.com/Lzy599775/agent-auto-sci-skills.git
cd agent-auto-sci-skills
.\scripts\install.ps1

推荐与 agent-auto-sci-data-viz、kdense-data-viz-selected 和 agent-auto-sci-scicomm 联合使用。

边界

  • 图件不能替代统计证据或方法解释。
  • 地图必须保留空间单位、CRS、图例、比例尺或必要的制图说明。
  • 不为美观牺牲数据可读性、标签完整性和 claim 准确性。
