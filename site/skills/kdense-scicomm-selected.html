K-Dense SelectedWritingCommunication

kdense-scicomm-selected

K-Dense 科学交流精选包，补强论文写作、同行评审、文献综述、引用管理、Mermaid 图示、slides、schematics 和 poster 输出。

Selected K-Dense scientific communication wrapper for writing, peer review, literature review, citation management, Mermaid, slides, schematics, and posters.

包含什么

论文与综述

scientific-writing、literature-review、peer-review、citation-management。

传播材料

scientific-slides、scientific-schematics、markdown-mermaid-writing、latex-posters、pptx-posters。

什么时候调用

使用 kdense-scicomm-selected 和 agent-auto-sci-scicomm，
帮我把体育地理综述稿件整理成投稿包：摘要、cover letter、图文叙事、claim-evidence map、审稿风险和回复矩阵。

Codex 部署

git clone https://github.com/Lzy599775/agent-auto-sci-skills.git
cd agent-auto-sci-skills
.\scripts\install.ps1

推荐作为 agent-auto-sci-scicomm 的工具增强，而不是替代领域写作判断。

边界

  • 不虚构引用和 DOI。
  • 不把一般写作润色当作期刊贡献。
  • 所有 claim 都必须能回到数据、文献或方法证据。
  • 海报和 slides 也要保留研究问题、证据和结论边界。
