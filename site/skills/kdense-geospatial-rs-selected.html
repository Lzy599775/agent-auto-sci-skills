kdense-geospatial-rs-selected

K-Dense 地理空间科学与遥感精选包，目前封装 GeoMaster 和 GeoPandas 两个上游 skill，用于补强空间数据处理、遥感预处理和 GIS 自动化说明。

Selected K-Dense geospatial and remote-sensing wrapper, currently packaging GeoMaster and GeoPandas.

包含什么

GeoPandas

矢量数据读取、CRS、空间连接、缓冲区、叠置、空间索引和可复现 GIS 脚本。

GeoMaster

综合地理空间工具链说明，适合遥感、空间数据和地理 AI 任务的入口型参考。

什么时候调用

使用 kdense-geospatial-rs-selected 和 agent-auto-sci-geospatial，
帮我检查体育设施可达性与绿地暴露项目中的 CRS、空间连接、缓冲区、栅格/矢量处理和可复现脚本结构。

Codex 部署

git clone https://github.com/Lzy599775/agent-auto-sci-skills.git
cd agent-auto-sci-skills
.\scripts\install.ps1

推荐先用 agent-auto-sci-geospatial 明确研究设计，再调用本 skill 补具体工具链。

边界

  • 工具链不能替代暴露定义、空间尺度和公平框架。
  • 遥感候选方法不能自动进入正式系统综述语料库。
  • 所有空间处理都需要记录 CRS、单位、空间连接规则和数据来源。
