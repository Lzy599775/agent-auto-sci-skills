Scientific Writing / Rebuttal / Presentation

agent-auto-sci-scicomm

科学交流与学术写作子 skill，用于论文结构、图文叙事、claim-evidence map、投稿信、审稿回复、PPT 和海报。

Scientific communication subskill for manuscript structure, figure-story alignment, claim-evidence maps, cover letters, rebuttals, slides, and posters.

适合什么时候用

论文写作与改稿

当已有结果但论文叙事松散、claim 不清楚或目标期刊定位不稳时使用。

审稿回复与展示

当需要 reply letter、response-to-reviewers、会议 PPT 或研究海报时使用。

典型输入

使用 agent-auto-sci-scicomm，帮我把结果部分重构为 claim-evidence map，并指出哪些图表、段落和政策建议需要加强。

Codex 部署与调用

安装整个仓库后重启 Codex。推荐调用方式：“使用 agent-auto-sci-scicomm，重构我的论文叙事、图文逻辑和审稿回复。”

.\scripts\install.ps1

它会输出什么

  • 论文结构、段落功能和图文叙事建议。
  • claim-evidence map 与缺口检查。
  • 投稿信、审稿回复和修改优先级。
  • PPT、海报和图表说明的表达框架。

使用注意

它不会替代实证证据。写作优化必须围绕已有数据、方法和结果展开，避免把语言润色变成证据扩张。
