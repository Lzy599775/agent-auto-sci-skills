agent-auto-sci-geospatial

地理空间科学与遥感子 skill，用于 GIS、矢量/栅格处理、CRS 审计、网络可达性、绿地暴露、热暴露和地图合规。

Geospatial subskill for GIS, vector/raster processing, CRS audits, network accessibility, green exposure, heat exposure, and map compliance.

适合什么时候用

空间数据处理

当研究涉及体育设施、公园、道路网络、人口网格、NDVI、LST 或 LCZ 数据时使用。

地图与暴露评估

当需要计算可达性、缓冲区暴露、网络距离、空间公平或绘制论文地图时使用。

典型输入

使用 agent-auto-sci-geospatial，审计我的体育设施可达性数据流程，重点检查 CRS、道路网络、服务半径、人口权重、MAUP 和地图合规。

Codex 部署与调用

安装整个仓库后重启 Codex。推荐调用方式：“使用 agent-auto-sci-geospatial，检查我的 GIS、遥感、暴露评估和地图合规流程。”

.\scripts\install.ps1

它会输出什么

  • 空间数据质量审计清单。
  • 可达性、暴露和公平指标设计。
  • 遥感指标、尺度效应和空间偏差检查。
  • 论文地图的图例、比例尺、投影、色带和合规建议。

使用注意

空间分析最容易因为 CRS、尺度、边界和网络假设产生隐性错误。该 skill 会优先检查这些基础问题，再进入复杂模型。
