K-Dense SelectedData AnalysisVisualization

kdense-data-viz-selected

从 K-Dense 中选择性封装的数据分析与科学可视化工具包。它服务于 EDA、统计、网络、并行数据处理和论文图表复现。

Selected K-Dense data-analysis and visualization wrapper for EDA, statistics, plotting, networks, parallel processing, and reproducible figures.

包含什么

分析基础

exploratory-data-analysis、statistical-analysis。

图表工具

matplotlib、seaborn、scientific-visualization。

复杂数据结构

networkx 支持网络关系，polars 和 dask 支持大表与并行处理。

本地适配

把分析输出连接到论文主张、补充材料、文献计量图和政策矩阵。

什么时候调用

使用 kdense-data-viz-selected，基于我的绿地暴露与体育设施可达性数据，
设计 EDA、统计检验、图表脚本和补充材料表格，并说明每张图支撑哪个 claim。

Codex 部署

git clone https://github.com/Lzy599775/agent-auto-sci-skills.git
cd agent-auto-sci-skills
.\scripts\install.ps1

推荐与 agent-auto-sci-data-viz 和 scipilot-figure-skill 联合使用：前者定论文图表逻辑，本 skill 补工具链，SciPilot 做视觉 QA。

边界

  • 不为了“复杂”而引入大数据工具。
  • 统计检验必须匹配数据结构、尺度和研究设计。
  • 图表必须有明确论文功能：回答问题、支撑 claim 或解除审稿疑虑。
