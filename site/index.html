快速选择

先看你现在的任务属于哪类，再进入对应 skill。复杂任务从总控开始，单点任务直接进入专项 skill。

总控

auto-sci-research

当任务跨越选题、文献、数据、分析、绘图、写作和投稿时，先让它拆路线。

空间

agent-auto-sci-geospatial

GIS、遥感、可达性、绿地/热暴露、空间公平、地图合规都先走这里。

综述

urban-exposure-review-radar-workflow

判断系统综述、范围综述、文献计量+批判性综述、前沿雷达或实证设计。

图件

scipilot-figure-skill

不知道用什么图、图不够投稿级、需要中文字体和视觉自检时使用。

写作

scipilot-writing-skill

润色、翻译、去 AI 味、cover letter、rebuttal 和写作质量机检。

geors-sci-writing-adapter

总控与自动化

用于拆任务、保留项目记忆、控制安全边界。

auto-sci-research

科研管线总控和 router。

输入
任务目标、已有材料、研究阶段。
输出
子 skill 顺序、路线图、质量门控。
边界
不替代具体分析和写作。
口令
使用 Auto-sci-research 规划这个科研任务。

agent-auto-sci-automation

source manifest、checkpoint、失败恢复和公开安全。

输入
数据源、文献源、目录结构、API 边界。
输出
manifest、状态表、恢复点、失败日志。
边界
不提交密钥、私有 PDF 或付费导出。
口令
建立这个项目的自动化科研流程。

方法论与综述路线

用于决定研究问题、证据等级、综述类型和期刊门槛。

agent-auto-sci-methodology

研究问题、机制、假设、偏倚和因果语言审计。

输入
选题、变量、草稿、结果。
输出
RQ 矩阵、机制图、证据分级、局限。
边界
不把相关结果写成因果。
口令
把这个选题改成可检验研究问题。

urban-exposure-review-radar-workflow

城市暴露、体育地理、遥感热环境和公共健康连接的路线判别。

输入
主题、目标期刊、语料来源、数据类型。
输出
route decision、journal gate、formal corpus/radar handoff。
边界
正式综述语料和前沿雷达候选必须分开。
口令
判断这篇综述/雷达/实证设计该走什么路线。

sport-geography-review-bibliometric

体育地理系统综述、scoping review 和文献计量。

输入
主题、数据库、关键词、目标期刊。
输出
检索式、PRISMA、编码表、图表计划、政策议程。
边界
文献计量必须服务机制框架或政策议程。
口令
设计综述 + 文献计量 + 批判性框架。

academic-research-suite

ARS 子 skill，处理深度研究、论文管线、审稿模拟和引用完整性。

位置
urban-exposure-review-radar-workflow/subskills
输出
研究收敛、文献综述、论文管线、审稿风险。
边界
只在父 skill 判定需要通用科研管线时调用。
口令
在城市暴露路线确定后，调用 ARS 做论文管线。

GIS、遥感与暴露

用于空间数据处理、暴露/可达性定义、遥感写作和技术 playbook。

agent-auto-sci-geospatial

GIS、遥感、可达性、绿地/热暴露、空间公平和地图。

输入
空间数据、指标、代码、地图或方法描述。
输出
CRS 审计、暴露窗口、可达性流程、地图检查。
边界
区分 exposure、accessibility、availability、quality、use。
口令
检查我的空间分析流程。

kdense-geospatial-rs-selected

K-Dense 精选 GeoPandas、GeoMaster 等 GIS/RS 技术参考。

输入
具体 GIS/遥感工具问题。
输出
库级别技术路线和注意点。
边界
不替代本地 CRS/尺度/暴露定义审计。
口令
调用 K-Dense 地理空间技术 playbook。

geors-sci-writing-adapter

地理/遥感 SCI 写作适配，覆盖反演、时空变化、暴露和可达性论文。

输入
论文类型、目标期刊、数据方法、要写章节。
输出
标题、摘要、IMRAD 段落、cover letter、写作检查。
边界
原创适配，不复制未授权上游参考全文。
口令
用 GeoRS 适配器重写这篇遥感论文的 Discussion。

sport-geography-sci-writing

体育设施、体育公园、暴露公平和城市健康实证论文写作。

输入
研究问题、方法、结果、目标期刊。
输出
期刊定位、gap chain、结果叙事、讨论和政策启示。
边界
先建立公共问题和学术缺口，不从技术指标开篇。
口令
把可达性/暴露结果写成 SCI 论文。

数据、图表与模型

用于 EDA、统计、机器学习、XAI、图件设计和可视化自检。

agent-auto-sci-data-viz

EDA、统计、文献计量图、论文图表和 caption。

输入
数据、结果、图表需求或已有图。
输出
figure plan、统计路线、caption、table shell。
边界
每张图必须服务一个 claim。
口令
设计主图、补充图和 caption。

agent-auto-sci-ai-ml

ML、SHAP/XAI、泄露检查、空间/时间验证。

输入
特征、标签、模型、验证方式、结果。
输出
baseline、切分方案、泄露清单、解释边界。
边界
SHAP 和特征重要性不是因果证据。
口令
检查模型验证和解释边界。

kdense-data-viz-selected

K-Dense 精选 EDA、统计、Matplotlib、Seaborn、NetworkX、Polars、Dask。

输入
库级别分析或绘图问题。
输出
技术路线、代码参考、工具注意点。
边界
不替代论文 claim 设计。
口令
调用 K-Dense 数据分析和可视化工具链。

kdense-ml-ai-selected

K-Dense 精选 scikit-learn、Lightning、Transformers、SHAP、TimesFM、GNN、UMAP。

输入
建模任务、技术栈、评估目标。
输出
模型路线、训练评估建议、解释方法。
边界
验证和结论强度仍由本地 ML/methodology skill 把关。
口令
调用 K-Dense ML 技术 playbook。

scipilot-figure-skill

科研数据图顾问，先选图再绘图并做视觉 QA。

输入
数据、论证目标、目标期刊、图件草稿。
输出
图型建议、绘图路线、色盲/裁切/字体检查。
边界
先判断图要证明什么。
口令
这份数据应该用什么图，帮我做投稿级图件。

写作、投稿与审稿

用于文本最后一公里、论文论证、投稿信、审稿回复和展示材料。

agent-auto-sci-scicomm

论文结构、claim-evidence map、投稿信、rebuttal、PPT 和海报。

输入
草稿、结果、目标期刊、审稿意见。
输出
IMRAD 大纲、审稿风险、回复矩阵。
边界
语言润色不能改变证据强度。
口令
重构论文 argument 和回复审稿人。

scipilot-writing-skill

写作与润色副驾驶，带 writing_lint 机检和 AI 读稿自审。

输入
文本、载体、目标期刊、语言方向、保守度。
输出
改后文本、回译、修改日志、lint 摘要。
边界
只动表达，不改事实、数值、引用和结论方向。
口令
用 SciPilot Writing 润色这段并做机检。

kdense-scicomm-selected

K-Dense 精选写作、审稿、引用、slides、posters、schematics。

输入
写作、引用、展示或示意图技术问题。
输出
模板、结构和工具参考。
边界
不替代领域贡献和期刊证据门槛。
口令
调用 K-Dense 科学交流技能包。
