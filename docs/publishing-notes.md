# Publishing Notes

Before publishing this repository:

1. Run `scripts/scan_public_safety.ps1`.
2. Confirm no private PDFs, raw WoS/Scopus exports, unpublished manuscripts, reviewer comments, or API keys are included.
3. Confirm local absolute paths have been replaced by generic placeholders.
4. Keep third-party repositories as references unless their license clearly permits vendoring.
5. Validate each `SKILL.md` if a local validator is available.
6. Review binary files manually before publishing.

Recommended GitHub repository names:

- `agent-auto-sci-skills`
- `sport-geography-agent-skills`
- `geo-sport-research-skills`
