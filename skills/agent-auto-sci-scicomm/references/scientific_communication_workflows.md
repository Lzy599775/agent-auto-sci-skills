# Scientific Communication Workflows

## 1. Manuscript Claim-Evidence Map

| Claim | Evidence | Figure/Table | Citation | Boundary | Section |
|---|---|---|---|---|---|

Write the map before drafting major sections.

## 2. Empirical Paper Structure

`public problem -> spatial inequity -> method/measurement gap -> framework -> decisive evidence -> mechanism -> planning implication -> boundary`

Use for `sport-geography-sci-writing`.

## 3. Review Paper Structure

`urban pressure -> fragmented knowledge -> transparent retrieval -> bibliometric/evidence map -> critical coding -> framework -> policy agenda`

Use for `sport-geography-review-bibliometric`.

## 4. Rebuttal Workflow

1. Split comments into: must fix, should fix, clarify, disagree with evidence, cannot do.
2. Identify underlying concern: novelty, rigor, evidence, method, writing, fit.
3. Match each response to a manuscript change.
4. Use Response / Revision / Location structure.
5. Avoid defensive tone.

## 5. Presentation Workflow

Slides should be:

- visual first;
- one claim per slide;
- minimal text;
- grounded in figures/tables;
- consistent with manuscript narrative.

Typical research talk:

1. Problem;
2. Gap;
3. Data and method;
4. Key results;
5. Mechanism;
6. Implication;
7. Limitations and next steps.
