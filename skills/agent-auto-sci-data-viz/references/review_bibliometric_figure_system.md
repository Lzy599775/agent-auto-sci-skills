# Review And Bibliometric Figure System

Use this reference when planning figures for systematic review, scoping review, bibliometric review, critical framework, policy agenda, or empirical sport-geography manuscripts.

## 1. Figure Families

| Figure type | Best use | Avoid |
|---|---|---|
| PRISMA flow | screening transparency | vague exclusion categories |
| search-strategy diagram | explain database/query logic | overcrowded query screenshots |
| annual publication trend | field evolution | raw counts without database boundary |
| journal/category map | venue distribution | treating journal count as quality |
| country/institution collaboration | geography of knowledge production | overclaiming collaboration strength |
| keyword co-occurrence | theme structure | calling clusters "causes" |
| thematic evolution | time-phased hotspots | arbitrary periods without rationale |
| citation/co-citation map | intellectual base | using citation count as impact alone |
| evidence map | method-topic-outcome coverage | decorative bubble charts with no coding basis |
| critical framework diagram | mechanisms and gaps | unsupported arrows |
| policy agenda matrix | actionable recommendations | generic recommendations detached from evidence |
| spatial map | exposure/accessibility/equity pattern | map without scale, CRS, or uncertainty |

## 2. Review/Bibliometric Core Set

For a high-standard Cities/SCS-style review, prefer:

1. PRISMA or transparent search/screening flow.
2. Corpus description figure or table.
3. Knowledge-structure visual: co-word, co-citation, or thematic map.
4. Evidence-map matrix: exposure/accessibility metric by population/outcome/scale.
5. Critical framework: mechanisms, blind spots, and policy translation.
6. Policy agenda: governance levers, data needs, and implementation risks.

## 3. Empirical Sport-Geography Core Set

For an empirical paper, prefer:

1. Study-area map with compliant locator and data layers.
2. Method pipeline diagram.
3. Exposure/accessibility distribution and equity-gap map.
4. Robustness/sensitivity figure for radius, travel mode, or metric choice.
5. Mechanism or policy-priority map/table.
6. Supplementary diagnostics: missingness, scale effects, model performance.

## 4. Policy Matrix Template

| Evidence gap | Affected population | Planning lever | Data needed | Risk if ignored |
|---|---|---|---|---|
| low sport-facility accessibility despite green exposure | older adults, youth, migrants | facility retrofitting, park programming, transit access | quality, opening hours, travel time | green-space policy overstates active-living benefit |

## 5. Caption Formula

Use this structure:

`What is shown -> data/source and scope -> how to read it -> key caution`

Example:

`Figure X shows neighborhood-level access to sport facilities within network travel-time thresholds. Darker values indicate higher cumulative opportunity after population weighting. Results should be interpreted as potential access, not actual facility use.`

## 6. Visual Consistency

- Use one palette family for one construct across the manuscript.
- Keep accessibility, exposure, vulnerability, and policy priority visually distinct.
- Use sequential palettes for magnitude, diverging palettes for gap/change, categorical palettes for themes.
- Do not reuse the same color for different constructs.
- Export both editable and publication-ready versions when possible.
