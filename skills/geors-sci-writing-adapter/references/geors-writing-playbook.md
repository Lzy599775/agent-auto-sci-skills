# GeoRS Writing Playbook

## 章节策略

| 章节 | 目标 | 必须交代 | 常见风险 |
|---|---|---|---|
| Title | 让研究对象、方法和空间/时间范围清晰可见 | 对象、区域、方法或机制、核心变量 | 标题只堆技术词，看不出问题意识 |
| Abstract | 用 5 步交代背景、目的、数据方法、关键结果、意义 | 研究缺口、样本/数据、方法、最重要数值、贡献边界 | 只描述流程，不报告发现 |
| Introduction | 从公共问题或学术缺口进入，再定位本研究 | 领域背景、现有不足、为什么该区域/对象重要、研究目标 | 用"随着...发展"开篇，缺少真实 gap |
| Methods | 让别人能复现空间和统计流程 | 研究区、数据源、时空分辨率、预处理、指标、模型/统计、验证 | CRS、尺度、暴露窗口、样本设计说不清 |
| Results | 按 claim 解释结果，而不是流水账 | 主发现、空间差异、时间变化、模型表现、稳健性 | 只报显著性或精度，不解释空间意义 |
| Discussion | 回到机制、治理、规划和文献对话 | 解释机制、与既有研究一致/不同、政策含义、局限 | 把相关说成因果，或重复 Results |
| Conclusion | 简洁归纳贡献和边界 | 关键发现、方法贡献、实践意义、未来方向 | 夸大普适性，新增未展示结果 |
| Cover letter | 告诉编辑为什么适合该刊 | 题目、研究问题、主要贡献、期刊契合、原创声明 | 像摘要复读，缺少期刊匹配理由 |

## 论文类型写法

### 遥感定量反演

重点写清四件事：

1. 为什么该属性需要遥感估计，而现有方法在区域、尺度、样本或变量上不足。
2. 数据链条：实测样本、遥感传感器、辅助变量、预处理、特征构建。
3. 模型链条：baseline、模型选择、训练验证切分、精度指标、不确定性。
4. 解释链条：哪些变量重要、空间外推到哪里有效、哪些区域或季节风险更高。

结果段建议顺序：模型总体表现 -> 变量/特征贡献 -> 空间分布 -> 误差与不确定性 -> 与已有研究对比。

### 时空变化分析

重点写清三层：

1. 变化事实：什么时候、在哪里、变化幅度多大。
2. 空间格局：热点、冷点、边界、梯度、城乡或区域差异。
3. 解释边界：驱动因素是统计关联、机制推断还是经过设计支持的因果解释。

避免把 Sen slope、Mann-Kendall、热点分析等工具当成贡献本身；贡献应落在过程理解、规划启示或机制识别上。

### 暴露、可达性与空间公平

必须区分：

- exposure：个体或群体在时空窗口内接触到什么环境状态。
- accessibility：从居住地或活动地到资源的机会成本。
- availability：资源是否存在或供应量是多少。
- quality：资源质量、服务能力或环境品质。
- use：人们是否实际使用以及使用频率。

写 Methods 时要说明空间单元、网络或欧氏距离、时间阈值、设施容量、人群权重、边界效应和敏感性分析。写 Discussion 时把发现连接到机会结构、公平性和治理工具，而不是只报告地图差异。

## 期刊定位

| 期刊族 | 更重视 | 写作提示 |
|---|---|---|
| RSE / ISPRS J P&RS / TGRS | 遥感方法、模型可靠性、验证严谨性 | 方法和不确定性必须细，贡献要超越应用案例 |
| IJRS / JAG | 遥感应用、区域过程、技术可复现 | 兼顾方法清晰和地学解释 |
| Catena / Geoderma | 土壤、地表过程、环境机制 | 讨论要回到地表过程和机制 |
| Cities / SCS | 城市治理、规划、空间公平 | 引言和讨论要突出城市问题与政策工具 |
| UFUG / Landscape and Urban Planning | 绿色空间、景观规划、人群收益 | 暴露、质量、可达性和使用要分开 |
| Environment International / Health & Place | 暴露-健康证据、混杂控制、人群异质性 | 因果语言和伦理边界要保守 |

## 投稿前自查

- 研究问题是否能用一句话说明。
- 所有变量是否有空间、时间和单位定义。
- 暴露、可达性、可用性、质量和实际使用是否被区分。
- 图表是否逐一支撑论文 claim。
- 统计或模型结果是否报告不确定性和验证方式。
- 讨论是否区分证据、推断和政策建议。
- 目标期刊近 3-5 年同类论文是否被用来校准结构和语气。
- 是否没有引入未核实文献、未公开数据或无授权内容。
