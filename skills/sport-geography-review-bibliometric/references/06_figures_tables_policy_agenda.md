# Figures, Tables, And Policy Agenda

Use this reference when designing review/bibliometric figures and tables.

## 1. Main Figure Logic

Every main figure must prove one claim.

Recommended figure set:

1. **Search and corpus profile**: PRISMA + year/journal/domain distribution.
2. **Knowledge evolution**: keyword overlay or thematic evolution.
3. **Method evolution**: method families by period.
4. **Justice and population coding**: stacked bars or heatmap.
5. **Conceptual framework**: object -> metric -> equity/justice -> health/activity -> planning.
6. **Policy agenda**: planning actions by scale and vulnerable group.

## 2. Tables

Recommended tables:

- search strategy and inclusion criteria;
- top journals / top cited papers / landmark references;
- coding framework and definitions;
- method families and limitations;
- research gaps and planning agenda.

## 3. Visual Rules

- Use consistent colours across comparable panels.
- Use sequential palettes for intensity and exposure.
- Use diverging palettes for inequity, gap, or difference.
- Use qualitative palettes for category groups.
- Do not overload VOSviewer screenshots with unreadable labels.
- Redraw summary figures in R/Python when journal polish is needed.

## 4. Planning Agenda Template

Translate findings into agenda items:

| Gap | Planning implication | Action scale |
|---|---|---|
| Static residential accessibility | include daily mobility and activity spaces | neighbourhood / city |
| Quantity-only green access | quality-adjusted access standards | park / neighbourhood |
| Distributional equity only | participatory and recognition-based planning | governance |
| Heat exposure ignored | integrate cooling accessibility and vulnerable groups | city / climate adaptation |
| Sports facilities isolated from parks | plan active recreation infrastructure networks | 15-minute city |

## 5. Policy Language

Prefer:

- "priority upgrading areas";
- "quality-adjusted access";
- "population-sensitive provision";
- "cooling service accessibility";
- "active recreation infrastructure";
- "participatory spatial governance";
- "evidence-informed planning standards".

Avoid:

- generic "government should pay attention";
- unsupported causal health claims;
- one-size-fits-all facility allocation.
